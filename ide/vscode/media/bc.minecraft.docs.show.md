# Show Microsoft Minecraft Docs

This command opens Microsoft's official Minecraft Bedrock documentation directly in your browser.

1. Open the command palette
2. Type in `Blockception: Show docs`
3. A filter box will appear, type in the name of the page you want to open
4. Press enter to open the page

## What the documentation covers

The Microsoft Minecraft Creator documentation includes reference material for:

- **Components** – Entity and block component definitions
- **Events** – Entity events and triggers
- **Molang** – Minecraft's expression language used in animations and rendering
- **Commands** – In-game slash commands and their syntax
- **Add-ons** – Guides and tutorials for creating behavior and resource packs

## Versioning

The documentation is versioned to match specific Minecraft releases. If you are targeting a particular version of Minecraft, make sure to select the correct version in the documentation's version picker to find the right information.
