# Workspace Symbols

Workspace symbols allow you to search for and navigate to symbols across your entire Minecraft Bedrock project workspace.

## How to Use Workspace Symbols

1. Open a Minecraft Bedrock project workspace
2. Press `Ctrl+T` (Windows/Linux) or `Cmd+T` (Mac) to open the workspace symbols search
3. Type the name or partial name of the symbol you're looking for
4. Select a symbol from the list to jump to its definition

## What Symbols Are Available

Workspace symbols search across all files in your workspace for:

### General Symbols
- **Fake Entities** - Entity references in behavior packs
- **Objectives** - Scoreboard objectives
- **Structures** - Structure definitions
- **Tags** - Entity and block tags
- **Ticking Areas** - Ticking area definitions

### Resource Pack Symbols
- **Animations** - Animation definitions
- **Animation Controllers** - Animation controller definitions
- **Attachables** - Attachable items
- **Block Culling Rules** - Block culling configurations
- **Entities** - Entity definitions
- **Fogs** - Fog definitions
- **Materials** - Material definitions
- **Models** - Model definitions
- **Particles** - Particle effects
- **Render Controllers** - Render controller definitions
- **Sounds** - Sound definitions
- **Textures** - Texture definitions

### Behavior Pack Symbols
- **Animations** - Animation definitions
- **Animation Controllers** - Animation controller definitions
- **Blocks** - Block definitions
- **Entities** - Entity definitions
- **Items** - Item definitions
- **Loot Tables** - Loot table definitions
- **Structures** - Structure definitions
- **Trading** - Trading definitions

## Tips

- The search is case-insensitive and supports partial matching
- Use the search to quickly find and navigate to any resource in your project
- Symbols are organized by their container (pack folder) for easier identification
