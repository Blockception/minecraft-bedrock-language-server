# Show Vanilla Minecraft Files

This command lets you browse the vanilla Minecraft Bedrock content bundled with the extension, including entities, items, blocks, and more.

1. Open the command palette
2. Type in `Blockception: Show vanilla file`
3. A filter box will appear, type in the name of the file you want to open
4. Press enter to open the file

## Why Browse Vanilla Files?

Viewing the vanilla Minecraft Bedrock files is a great way to understand how the game is structured. You can use them as a reference for available components, their properties, and how content is organized — making it easier to create your own custom content.

> **Note:** The vanilla files are read-only references bundled with the extension. They cannot be edited directly.

## Tip

Use the vanilla entity files as a starting point when creating custom entities. By examining how vanilla entities define their components and behaviors, you can copy patterns and adapt them for your own creations.
