# Document Symbols

Document symbols provide a quick way to view and navigate to all symbols defined in the current file.

## How to Use Document Symbols

1. Open a Minecraft Bedrock project file (e.g., a JSON file containing entities, objectives, or other resources)
2. Press `Ctrl+Shift+O` (Windows/Linux) or `Cmd+Shift+O` (Mac) to open the document symbols dropdown
3. Start typing to filter symbols by name
4. Click on a symbol or press Enter to jump to its location in the file

## What Symbols Are Available

Document symbols include:
- **Fake Entities** - Entity definitions in behavior packs
- **Objectives** - Scoreboard objectives
- **Structures** - Structure definitions
- **Tags** - Entity and block tags
- **Ticking Areas** - Ticking area definitions
- **File Names** - For mcfunction files

## Alternative Access

You can also access document symbols through:
- The Outline view in the Explorer sidebar
- The breadcrumb navigation at the top of the editor
